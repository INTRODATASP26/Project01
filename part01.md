Reflections - project 1

I was between that and extracting my AO3 data, which is another website I use a lot, but I didn't want to expose myself too much.

I did struggle around figuring out the logic to setup (especially because I chose to start a project on VScode this time and realized I had to download and setup a lot of modules), but having the logic down the process was a bit more striaght forward. Not doing math, yay! I end up using Claude as an aid, but I did try to at least pseudocode and get errors for two or three times before going for help.

The process was fun!
